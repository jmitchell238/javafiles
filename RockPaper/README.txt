PROJECT TITLE:			Rock, Paper, Scissors Game

PURPOSE OF PROJECT:		Assignment for Computer Science Class

VERSION or DATE:		v0.1

HOW TO START THIS PROJECT:	CMD Prompt> File Folder> -$ java RockPaperJamesM.java

AUTHORS:			James Mitchell

USER INSTRUCTIONS:		The user will be asked to enter 1, 2, or 3 for their choice of 
					rock, paper, or scissors.

OUTPUT:				The program will inform the user whether they have won or lost
					against the computer. It will then ask them if they 
					would like to play again. This will continue until
					the user decides not to play any longer.
