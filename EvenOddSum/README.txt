PROJECT TITLE:			Even Odd Sum

PURPOSE OF PROJECT:		Assignment for Computer Science Class

VERSION or DATE:		v0.1

HOW TO START THIS PROJECT:	CMD Prompt> File Folder> -$ java EvenOddSumJamesM.java

AUTHORS:			James Mitchell

USER INSTRUCTIONS:		User will enter an Integer and then select (y/n) to continue.
				User may enter as many numbers as they would like.

OUTPUT:				Once the User selects (n) to not continue, the program will
				then show them the Sum of the even numbers they inputed
				and the Sum of the odd numbers they inputed.
