PROJECT TITLE:			Employee And Production Worker Classes

PURPOSE OF PROJECT:		Assignment for Computer Science Class
				This program is an example of creating constructors,
				accessor and mutator methods.

VERSION or DATE:		v0.1

HOW TO START THIS PROJECT:	CMD Prompt> File Folder> -$ javac *.java
							 -$ java WorkerDemoJM.java

AUTHORS:			James Mitchell

USER INSTRUCTIONS:		This program contains three files. EmployeeJM.java, ProductionWorkerJM.java,
				and WorkerDemoJM.java. The user will compile the program with the first CMD
				instruction above. The user can them run the program with the second CMD
				instruction above.

OUTPUT:				The program will display in the CMD prompt information about production workers
				created in the source code.
