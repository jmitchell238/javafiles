PROJECT TITLE:			Mobile Service Provider

PURPOSE OF PROJECT:		Assignment for Computer Science Class

VERSION or DATE:		v0.1

HOW TO START THIS PROJECT:	CMD Prompt> File Folder> -$ java MobileServiceProviderJamesM.java

AUTHORS:			James Mitchell

USER INSTRUCTIONS:		The user will be asked to Enter which Service Package they have
				based on the information given in the program assignment insturctions.
				The user will then enter the amount of minutes they used in previous month.

OUTPUT:				The user will be presented with a Message that restates which Package
				the user has, The minutes they exceeded over the package plan minutes allowed.
				It will also inform the user the extra charge the user was charged and the 
				total charge the user is responsible for.
